# Daily Standup

Generate a daily standup summary from Obsidian vault.

## Instructions

Run the daily standup agent to show:
- Yesterday's completed items across all projects
- Today's next steps
- Current blockers

Execute this command:

```bash
python3 ~/agents/daily-standup/standup.py
```

For weekly summary:

```bash
python3 ~/agents/daily-standup/standup.py --week
```

Display the output to the user.
